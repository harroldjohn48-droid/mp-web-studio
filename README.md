# M&P Web Studio

A professional web design and development studio website built with vanilla HTML, CSS, and JavaScript.

## Features

- Responsive, mobile-first design
- Sticky navigation with mobile hamburger menu
- Hero section with animated background blobs
- Services overview grid
- Portfolio / case studies section
- About section with team stats
- Client testimonials
- Contact form with client-side validation
- Scroll-driven fade-in animations (IntersectionObserver)
- Dark theme with CSS custom properties

## Project Structure

```
mp-web-studio/
├── index.html       # Main HTML page
├── css/
│   └── style.css    # All styles
├── js/
│   └── main.js      # Interactive behaviour
└── README.md
```

## Getting Started

No build tools required — open `index.html` directly in any modern browser, or serve the folder with any static file server:

```bash
npx serve .
```
